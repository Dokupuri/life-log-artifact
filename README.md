# 麻雀 精算

各自の**持ち点**を入れるだけで、**順位・ウマ・オカ・ビンタ・焼き鳥・チップ・ソシー**まで自動計算する
麻雀スコア精算アプリ（スマホ最適化・単一HTML・オフライン動作）。ルール違いの複数ゲームを1日分まとめて集計できる。

**公開URL**: https://dokupuri.github.io/life-log-artifact/
スマホのブラウザで開き「ホーム画面に追加」でアプリのように使えます。

## 使い方
- **ゲーム切替バー（上部）**: `＋ゲーム` でルール違いのゲームを追加。ルール（配給原点・返し点・ウマ・ビンタ・焼き鳥・チップ・ソシー）はゲームごとに独立。
- **入力**: 各自の持ち点をそのまま入力（`80000`ならそのまま）。焼き鳥ありの半荘はチェック。3人入れると4人目を自動計算。
- **記録**: このゲームの累計＋半荘ごとの記録（焼き鳥表示・編集・削除）。
- **精算**: 最終チップ・借チップ → チップ差 → チップ込み合計 → ソシー。
- **合計**: ルール違いの全ゲームを合算した1日の総計。結果コピー・1日の保存・履歴。
- **設定**: メンバー（人数・名前・プリセット）は1日共通。入力単位（そのまま/100点）。ルールはゲームごと。規定値の保存/リセット。

## 計算（丸めなし・表示時のみ千点換算）
1. 順位 = `RANK(持ち点, 全員, 降順)`
2. 素点 = 持ち点 − 返し点
3. オカ = `(返し点 − 配給原点) × 人数` を1位へ
4. 順位ウマ = 順位→ウマ表
5. ビンタ = 単価×(下位人数−上位人数) ± 単価×(浮き/沈み相手人数)
6. 焼き鳥 = 各焼き鳥が単価を他の非焼き鳥全員へ（ゼロサム）
7. 総得点(半荘) = 素点+オカ+ウマ+ビンタ+焼き鳥 → 累計 ÷1000
8. チップ差 = 最終チップ−初期チップ−借チップ／チップ込み合計 = 総得点 + チップ差×(点/チップ)/1000／ソシー = チップ込み合計 × 倍率

## 技術
